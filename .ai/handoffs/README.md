- Один файл на задачу: `PTR-XXX-<kebab-slug>.md`
- Формат — см. [`.ai/rules/70-orchestration-codex-cli.md`](../rules/70-orchestration-codex-cli.md)
- Файлы коммитятся в репо вместе с кодом задачи и **не удаляются** после merge
